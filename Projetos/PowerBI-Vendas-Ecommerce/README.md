# Power BI – Vendas Ecommerce

Projeto desenvolvido em Power BI com foco na análise de vendas de e-commerce
e construção de dashboards interativos.

![Dashboard de Vendas de E-Commerce](./dashboard-finalizado/imagem-dashboard.png)

## Objetivo
Analisar o desempenho de vendas por região e período, utilizando indicadores
e visualizações claras para apoio à tomada de decisão.

## Principais Análises
- Faturamento total
- Vendas por região
- Evolução das vendas ao longo do tempo

## Habilidades Demonstradas
Este projeto demonstra a aplicação prática de processos de ETL (Extract, Transform, Load)
no Power BI para análise de dados de vendas. As principais habilidades desenvolvidas incluem:

- **Conexão com datasets:** Importação e integração de dados a partir de arquivos CSV,
  organizando informações para análises por região e período.
- **Power Query Editor:** Utilização do editor para limpeza e preparação dos dados,
  incluindo remoção de inconsistências e padronização de informações.
- **Transformações de dados:** Aplicação de técnicas como mesclagem de tabelas,
  tratamento de erros e reorganização de colunas para facilitar a análise.
- **Modelagem de dados:** Criação de relacionamentos entre tabelas (produtos, vendas e regiões),
  estruturando um modelo adequado para dashboards interativos.
- **Visualização de dados:** Construção de gráficos e indicadores para análise de faturamento
  e evolução das vendas ao longo do tempo.

## Observações
