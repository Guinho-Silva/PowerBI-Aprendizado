# PowerBI - Vendas Globais

Projeto desenvolvido em Power BI com foco na análise de vendas globais
e construção de dashboards interativos.

## Objetivo
Analisar o desempenho de vendas por região e período, utilizando
indicadores e visualizações claras.

## Principais Análises
- Faturamento total
- Vendas por região
- Evolução das vendas ao longo do tempo

## Observações
Dados fictícios ou educacionais, utilizados para fins de estudo e portfólio. Projeto desenvolvido com base em curso da plataforma **Data Science Academy** módulo 2 com documentação própria.
