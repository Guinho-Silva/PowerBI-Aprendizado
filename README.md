# Projetos em Power BI

Este repositório reúne projetos desenvolvidos em Power BI com foco em análise
de dados e construção de dashboards interativos.

Cada projeto possui um objetivo específico de aprendizado, abordando diferentes
tipos de análise e visualização de dados.
