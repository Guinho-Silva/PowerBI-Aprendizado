# Projetos em Power BI

Este repositório reúne projetos desenvolvidos em Power BI com foco em análise
de dados e construção de dashboards interativos.

Cada projeto possui um objetivo específico de aprendizado, abordando diferentes
tipos de análise e visualização de dados.

## Projetos e Foco de Aprendizado

### PowerBI-Vendas-Globais
- Análise de vendas por região
- Comparação de faturamento por período
- Criação de métricas de desempenho

### PowerBI-Vendas-Ecommerce
- Análise de vendas online
- Avaliação de produtos mais vendidos
- Visualização de tendências de consumo

### PowerBI-Vendas-Livraria
- Análise de vendas por categoria de livros
- Comparação de faturamento
- Organização de dados por período

### PowerBI-Vendas-Petshop
- Análise de vendas por tipo de produto
- Indicadores de desempenho
- Visualização de dados de forma clara

## Ferramentas Utilizadas
- Power BI Desktop

## Observações
Os dados utilizados são fictícios ou educacionais.
Alguns projetos foram desenvolvidos com base em cursos e plataformas de estudo,
com os devidos créditos informados nos READMEs individuais.
